# StreamAI - Creator Monetization Platform

A full-stack streaming platform where creators upload content,
AI automatically generates transcripts and summaries, and 
viewers subscribe or pay per video

## Tech Stack
- Next.js 16 (App Router)
- PostgreSQL + Prisma 7
- OpenAI API (Whisper + GPT-4)
- Stripe Connect (marketplace payments)
- AWS S3 + CloudFront (video storage + CDN)
- JWT Authentication

## Features 
- [ ] Video upload with S3 storage
- [ ] AI transcript generation (OpenAI Whisper)
- [ ] AI content summarization (GPT-4)
- [ ] Stripe subscription plans
- [ ] Pay-per-view content
- [ ] Creator dashboard with analytics 
- [ ] Role-based access (viewer/creator/admin)
- [ ] Full-text search
